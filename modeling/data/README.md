- Source: `*_exp` is the experimentally measured values, `*_mod_data` is output values from the data-driven model, and `*_mod_physical` is output values from the physics-based model.  
- Subsystem ID: `cc` for combined cooler, `sc` for surface condenser, `dc` for dry cooler (ache type) and `wct` for wet cooling tower.
