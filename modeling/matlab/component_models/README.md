- [wct_model_data.mat](./wct_model_data.mat) and [dc_model_data.mat](./dc_model_data.mat) are the GPR trained models using experimental data from the pilot plant. Limited in range but light.
- [model_data_wct_fp_gaussian.mat](./model_data_wct_fp_gaussian.mat) and [model_data_dc_fp_gaussian.mat](./model_data_dc_fp_gaussian.mat) are trained using a lot of data evaluated using the first-principles model for the pilot plant. Quite heavy model, specially important for the WCT (~16MB).

More models can be found in the [modeling/data](./../../data/models_data/).
